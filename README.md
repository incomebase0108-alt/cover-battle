# Cover Battle — 岩と森の 4 vs 4 バトル

ブラウザだけで遊べる、見下ろし型の 4 対 4 タクティカルバトルです。
岩を盾にし、森に隠れながら、敵チームを全滅させてステージを攻略しよう。
（スプラトゥーンのような「地形を使った撃ち合い」をシンプルにしたゲームです）

## 遊び方

`index.html` をブラウザで開くだけ。インストールやビルドは不要です。

```bash
# そのままダブルクリックで開くか、ローカルサーバーでもOK
python3 -m http.server 8000
# → http://localhost:8000 を開く
```

## 操作

| 操作 | キー |
|---|---|
| 移動 | `W` `A` `S` `D` または 矢印キー |
| 照準 | マウス |
| 射撃 | 左クリック または スペース |
| 爆弾設置 | `E`（または `Q` / `Shift`） |

## ルール

- **青チーム（4人）** を率いて **赤チーム（4人）** と戦う。あなたは青の1人を操作し、残りの味方3人はAIが担当します。
- 相手チームを **全滅** させるとステージクリア。味方が全滅すると敗北。
- ステージは全3面。クリアするごとに敵が強くなります。

## 地形

| 地形 | 効果 |
|---|---|
| 🪨 **岩** | 弾と移動をブロックする盾。撃ち続けると **破壊でき**、壊すと **アイテム** が出ることがある。 |
| 🌲 **森** | 中に隠れると **敵からは見えなくなり狙われない**（味方と自分には見える）。敵が近づいて来ると見つかる。 |

## アイテム（破壊した岩からドロップ）

| アイテム | 効果 |
|---|---|
| **HEAL** | 体力を回復 |
| **SPEED** | 移動速度アップ |
| **B.SPD** | 弾のスピードアップ |
| **RANGE** | 弾の飛距離アップ |
| **BOMB+** | 同時に置ける爆弾の数アップ |

## 爆弾（ボンバーマン風）

- `E` キーで足元に設置。少し待つと爆発します。
- 爆風は **岩を一気に破壊** し、範囲内のユニットに大ダメージ。
- **味方や自分も巻き込む** ので、設置したら離れること！

## 構成

| ファイル | 役割 |
|---|---|
| `index.html` | 画面・メニュー・スクリプト読み込み |
| `css/style.css` | スタイル |
| `js/vector.js` | ベクトル/幾何ヘルパー（距離・線分と円の判定など） |
| `js/config.js` | パラメータ定数とステージ定義 |
| `js/input.js` | キーボード・マウス入力 |
| `js/map.js` | 地形（岩・森）の当たり判定と描画 |
| `js/entities.js` | ユニットと弾 |
| `js/ai.js` | 味方・敵AIの行動 |
| `js/game.js` | ゲームループ・状態管理・勝敗判定 |
| `js/ui.js` | HUD・メニューの表示制御 |
| `js/main.js` | 起動とボタン配線 |

## デプロイ（外部サーバーへの公開）

静的サイトなのでどこにでも置けます。手順は [`DEPLOY.md`](DEPLOY.md) を参照。
`main` に push すると GitHub Actions（`.github/workflows/deploy-pages.yml`）で
GitHub Pages に自動公開されます（Settings → Pages → Source を「GitHub Actions」に設定）。
